File view

TagScanner has usefull features for filelist customization.

You can setup Sorting and Grouping files in list with predefined or user-defined modes.

With RightClick on columns you can change list of visible columns and also add user-defined columns to view.

Add column Runs add column wizard
Edit Change options of current user-defined column
Choose columns... Set visibility of columns
Reset to default Reset columns to initial state
Autofit by content Set columns width according to their content

With RightClick menu on list you can do additional operations with tags and files :

Play Playback current file with internal player
Open Open file with default application
Open file location Find file in Windows Explorer
Autonumbering Set Track/Disc value for selected files with format: counter/number of files
TAG Cut Cut tag data from selected files into program buffer
TAG Copy Copy tag data from selected files into program buffer
TAG Paste Paste tag data consistently from program buffer into selected files
Rename folder by TAG Rename current file folder with predefined or user template
Imprort cover from file Import covers from specified files into tags within selected files
Export cover to file Export covers from selected files into specified files
Import tags from CUE file Fill in tag data from cue file containing a list of tracks
Convert codepage Convert unreadable text from expected encoding to Unicode
Remove Remove selected files from list
Move to Move selected files to the specified folder
Copy to Copy selected files to the specified folder
Rename Rename selected file
Delete Delete selected files from disc

DoubleClick on file to listen it or on group title to select all files in this group.

A filename highlighted in blue indicates that the file path exceeds basic system limits and in some cases cannot be processed correctly by other applications.
