ONLINE

Searching for album information and covers.

Send request to prefered service and select desired album from list. You can unselect individual tracks if they are not needed.

Click Save to import data into selected files.

Freedb, Tracktype

These online services allow to get only textual information and only for one disc at one time.
Is it recommended to use these services only for automatic search the related data on selected tracks. For manual search it is recommended to use other services.

You can choose between different methods for getting information:
Calculate DiscID from selected files Select all tracks from only one album in correct order.
Search in database Enter artist or album name in Search box.
Manual query Enter a DiskID by yourself. You can find a DiskID by using the search function at freedb.org.

Click Search to make request to service.

If you have problems with displaying of received album info, click Convert codepage and choose the appropriate codepage from the menu.

If you want to write Artist name instead 'VA' into Artist tag for compilations, just check Mark as compilation in menu.

Discogs, MusicBrainz

These services allow to get text information and cover art for all types of media and support multidisc albums and compilations.

Enter lookup text and click Search to run searching.
Also you can search for necessary album in browser and then enter album ID in search box to get data without searching.
