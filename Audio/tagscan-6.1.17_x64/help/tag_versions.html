TAGs information

Program supports ID3 1.0/1.1/2.2/2.3/2.4 tags, APE v1 and v2 tags, Vorbis Comments, WMA tags and MP4(iTunes) metadata. Some files may contain several tags at the same time.

Table of combinations of file formats and tags:

¹ Update ID3v1 tag only if it exists before (by default)
² Update APEv2 tag inside MP3 files only if it exists before

You can change tags reading priority in program options.
Program reads tag-information only once and not refreshing it when you switched to another tags reading mode, so press Ctrl + F5 to update view.

Program never deletes any unsupported tag fields.

All tags except ID3v1, ID3v2 and APEv1 always saved in Unicode. ID3v1 and APEv1 tags don't support Unicode at all and ID3v2 tags can optionally save Unicode data (ON by default).
If you interact with programs, portables or car audios which have problems with ID3v2 tags stored in Unicode just uncheck ID3v2 unicode saving option.

ID3v2 tags stored in beginning of the file, so program needs to move all file data and that's why write process may take too much time at first time. OGG, WMA, MP4 files are recreated anew in most cases, that's why write process may take appreciable time for every change.
