EDIT

Edit text information and embedded covers.

Select files to edit using the mouse or keyboard. The metadata from selected files will be displayed in the entry fields on the right side.
If field contains text < varies >, the program will keep the current content of this field when saving the new tag.

Fill in the entry fields.

Click on field name to lock value in edit box, this field value will not changed when selecting another file.

Click Save button to save new tags into the selected files.

TagScanner shows version and type of readed tag for current file.
