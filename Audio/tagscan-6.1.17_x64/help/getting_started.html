Getting started

Start with selecting folder with music files or dragging files and folders into left pane. TagScanner will read the metadata from all supported files and show it according to the current grouping and sorting mode.
If you want to display files in subfolders, check the option Search subfolders

Now select the files you want to process by using mouse or keyboard and switch to necessary module.

For more information, see the sections describing the use of a each module:
RENAME Rename files based on the tag and file information
EDIT Edit Tags, add covers
ONLINE Import tags and covers from online services
GENERATE Fill, change and format tag information in different ways
EXPORT Create playlists and export infromation about your music collection to any format
